RAG Challenge / RAG 挑战

TKE Interview Lab hosts a technical assessment challenge for building a production-grade RAG (Retrieval-Augmented Generation) system. Candidates must implement a full-site knowledge base chatbot with authentication, deploy it publicly, and submit code for review.

TKE 面试实验室提供一个技术评估挑战，用于构建生产级 RAG（检索增强生成）系统。 候选人必须实现带认证的全站知识库聊天机器人，公开部署，并提交代码供审查。
